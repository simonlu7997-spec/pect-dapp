# PECT DApp 网站优化建议分析

## 优化需求总结

### 1. 取消首页"双代币介绍"下的两个按钮
- 移除"购买"页面增加的"加入白名单"按钮
- 移除 PV-Coin 和 C2-Coin 卡片下的"加入白名单"按钮

### 2. 首页中部"加入白名单"权益列表改为：
- ✓ 满足监管机构要求
- ✓ 优先购买 PV-Coin 和 C2-Coin
- ✓ 专属投资者社区和资讯
- ✓ 定期收益报告和市场分析

### 3. 首页底部"产品"下面按按钮序改为：
- "代币购买"、"收益分红"和"质押奖励"
- 分别对应"购买"、"资产"、"质押"三个页面

### 4. 首页底部"资源"下面"支持"、"常见问题"、"联系我们"应别对应三个页面
- "文档"页面放入项目白皮书中文版和英文版
- "常见问题"页面列出一些常见的问题和解答
- "联系我们"页面列出项目的几个常用联系方式

### 5. 在首页"绿色资产×数字金融"的右边图片栏中轮流显示三个电站的现场照片
- 点击照片可以显示三个电站的实时视频
- 并在图片栏下增加"电站实时视频"几个字

### 6. 首页"绿色资产×数字金融"下面"了解更多"按钮应链接"文档"页面

### 7. 在首页"实时数据面板"后增加"电站 A"、"电站 B"、"电站 C"三个按钮
- 点击进去后显示该电站 7 天、30 天、近 1 年的发电量（数据来源：国网公司 APP）

### 8. 首页"实时数据面板"下面，"发电量"改为"总发电量（2026 年）"
- "电费收入"改为"总电费收入（2026 年）"
- "分红池"改为"分红池（2026 年）"
- 数据每月手动更新一次

### 9. 首页"电站资产分组"下面表格，"发电量"改为"年发电量"，"收入"改为"年收入"

### 10. 首页"代币分布"中，"公募"改为 50%，"生态基金"改为 5%，"团队"改为 10%，"流动性"改为"项目方储备"

### 11. 首页"加入白名单"右边图片栏，去网上搜一张漂亮的光伏电站图片贴上
